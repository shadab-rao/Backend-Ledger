# Backend Ledger System

A secure backend ledger system built with **Node.js**, **Express.js**, and **MongoDB** that implements double-entry bookkeeping for financial transactions. The project provides user authentication, account management, transaction processing with idempotency support, immutable ledger entries, balance calculation using aggregation, and role-based authorization for system operations.

### Features

* User authentication with JWT
* Account creation and management
* Double-entry ledger (Credit/Debit)
* Transaction processing with idempotency
* Immutable ledger records
* Real-time account balance calculation
* MongoDB transactions for data consistency
* Role-based access for system/admin operations
* Email notifications for transactions

### Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Nodemailer
